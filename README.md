# HydraStart
Simple yet useful restarter plugin for your Minecraft PE server [PocketMine-MP] !

## Features

> Auto reconnecter [Makes player(s) automatically rejoin **beta**]

> CountDown system

> Fully customizeable messages

> No overloading

# Branches
